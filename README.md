## Rodando localmente

Clone o projeto

```bash
  git clone git@github.com:Taunt-byte/projeto-integrador2.git
```

Entre no diretório do projeto

```bash
  cd projeto-integrador2
```

Instale as dependências

```bash
  npm i
```

Inicie o servidor

```bash
  npm run start
```
